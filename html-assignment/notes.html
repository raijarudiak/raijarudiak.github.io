5 HTML Tags I've Learnt and Their Descriptions

h1 is one of the heading tags.
Heading tags range between h1 and h6, with h1 being the biggest and h6 being the smallest. These are used for highlighting important topics, with the most important being the biggest. This is shown above, however I used h3.
p is the paragraph tag.
This commences the body text (paragraphs). Browsers will create a margin (one line of blank text) before and after each p tag.
ol is the ordered list tag.
This will create a list of items in order, numbered. This would be a good tag to use to create step-by-step instructions, for example. This list of tag definitions is also an example of this.
ul is the unordered list tag.
This will create a list of items in no specific order, with ticks. This would be a good tag to use to list off some of your hobbies, for example. This is demonstrated on the previous page.
sup is the superscript tag.
This will superscript the text, meaning half a character above the normal line and is usually smaller than the normal text. This is a good tag to use when writing the date. For example: October 7th.
